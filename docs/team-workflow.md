# Team Workflow

## Board columns

- `Backlog`
- `Todo`
- `In Progress`
- `Review`
- `Done`

## Suggested roles

- Person 1: picks implementation tasks
- Person 2: reviews pull requests
- Swap roles every few tasks

## Simple sprint simulation

1. Create 5 to 8 issues.
2. Move 2 to `Todo`.
3. Each person takes one issue.
4. Open a pull request for each finished task.
5. Review each other and merge.
